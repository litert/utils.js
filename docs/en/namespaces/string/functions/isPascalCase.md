# Function: `isPascalCase()`

> **Source:** [NameCase.ts](https://github.com/litert/utils.js/blob/master/packages/partials/string/src/Functions/NameCase.ts)

## Import

```ts
import { isPascalCase } from '@litert/utils-string';
```

## Signature

```ts
const isPascalCase: typeof isUpperCamelCase;
```

## Description

Alias for [`isUpperCamelCase()`](./isUpperCamelCase.md). PascalCase and UpperCamelCase refer to the same naming convention.

Returns `true` if the input starts with an uppercase letter and contains only letters and digits.
